# File Map

* `css/`: Compiled CSS files from `scss/`
* `js/`
    * `_*/`: Javascript for `*.html`
        * `main.js`: Main interaction logic
        * `ng.js`: AngularJS controllers
    * `depend/`: Dependencies
        * `angular-1.6.6`: AngularJS 1.6.6 Framework
* `scss/`: Raw SCSS files, edit these to see changes
* `www_html`: HTML files
